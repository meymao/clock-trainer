# 🕐 clock-trainer

Interactive analog clock app for kids to learn how to tell time.
Supports Japanese 🇯🇵 and English 🇬🇧 — switchable with one tap.

子ども向けのインタラクティブなアナログ時計アプリです。
日本語🇯🇵・英語🇬🇧をワンタップで切り替えられます。

---

## ✨ Features / 機能

- Draggable hour and minute hands
- Minute-by-minute number display
- Voice readout in Japanese and English, auto-selecting the best available voice per browser
- Responsive layout for iPad and tablet use
- No installation required — runs in any browser

&nbsp;

- 短針・長針をドラッグして動かせる
- 1分刻みの数字表示
- 日本語・英語の音声読み上げ（ブラウザに応じて最適な音声を自動選択）
- iPad・タブレット向けのレスポンシブレイアウト
- インストール不要 — ブラウザだけで動く

## ⚠️ Note on Japanese Pronunciation / 音声読み上げについて

The Japanese voice readout uses the browser's built-in text-to-speech engine, which may mispronounce certain words — for example, reading "ごじゅっぷん" as "ごじゅうぷん". We recommend parents listen along and gently correct any inaccuracies as a natural part of the learning experience.

日本語の読み上げはブラウザ内蔵の音声合成エンジンを使用しているため、一部の読み方が不正確になる場合があります（例：「ごじゅっぷん」が「ごじゅうぷん」と読まれるなど）。お子さんと一緒に使いながら、保護者の方が適宜フォローしていただけると幸いです。
